# 📰 GNews RSS App

A modern full-stack news application that aggregates real-time news using RSS feeds, built with React (Vite), Tailwind CSS, and Node.js.

---

## 🚀 Features

### 📰 News System

* Fetch real-time news from RSS feeds
* Category-based filtering (India, World, Business, Technology, etc.)
* Search news by keywords
* Detailed news view with source links

### 🔐 Admin Panel

* Secure admin login
* Create and manage news
* Control and update content dynamically

### 📢 Advertisement System

* “Advertise With Us” page
* Dedicated ad sections in UI
* Scalable ad placement support

### 🎨 UI/UX

* Built with Tailwind CSS
* Fully responsive design
* Clean and modern interface
* Fast performance with Vite

---

## 🛠 Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js

### Other

* RSS Parser
* REST API

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash id="r0z3vl"
git clone https://github.com/dellibabus/GNews.git
cd GNews
```

---

### 2️⃣ Install Dependencies

```bash id="m9d0ey"
# Frontend
cd Frontend
npm install

# Backend
cd Backend
npm install
```

---

### 3️⃣ Run Application

```bash id="y92kqz"
# Backend
cd Backend
npm run dev

# Frontend
cd Frontend
npm run dev
```
